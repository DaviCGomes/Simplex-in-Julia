Implementação do Simplex utilizando o método tableau na linguagem Julia.
A função objetivo está no arquivo "trablaho.txt". O programa realizará a leitura desse arquivo e tentará buscar a solução ótima fazendo manipulações na tabela.
O resultado obtido será o vetor contento a Base, as matrizes 'c', 'b' e 'A' manipuladas. O programa, também, mostra o dual e a sensibilidade para os valores contidos na matriz 'b'.
